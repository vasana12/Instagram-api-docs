Request trial . This may be used to get real instagram account session and cookie data

Parameters

  • url (str) – Description of url.

  • post (str, optional) – required data when you call POST method.

  • with_signature (bool, optional) – if with_signature, send_request excute generate_signature.

  • extra_sig (bool, optional) – if extra_sig, send_request add this to post with “&”.

  • timeout (int, optional) – set request timeout
